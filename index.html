Revisão de Programas PPG

Acompanhamento do processo de revisão das disciplinas — uso exclusivo admin

📖 Metodologia de Revisão

Esta seção reúne o processo de revisão das disciplinas dos programas de pós-graduação da UNEMAT. Os dados refletem a análise qualitativa das ementas, objetivos e pertinência temática de cada componente curricular.

Status:

✅ Aceito — disciplina aprovada sem necessidade de alteração.
❌ Rejeitado — disciplina recomendada para substituição ou exclusão.
⏳ Pendente — em análise ou aguardando documentação.
🆕 Sem revisão — ainda não avaliada.

As notas administrativas são salvas localmente no seu navegador e visíveis somente para o admin.

Sobre o HUB da Pós

Informações e créditos da plataforma

🎓 O que é o HUB da Pós?

Esta plataforma reúne dados de projetos institucionalizados na UNEMAT, extraídos do sistema SIGAA, a partir de 2024. Permite visualização, filtragem e análise cruzada de atividades de pesquisa, extensão e pós-graduação stricto sensu em todos os campi.

🔗 Novidades desta versão integrada

Esta versão Admin inclui: (1) Integração P&E — identificação automática de docentes com atuação simultânea em pesquisa e extensão, com sugestão de programas integrados; (2) Revisão de Programas — acompanhamento do processo de revisão curricular dos PPGs; (3) Sistema de notas admin — anotações privadas salvas no dispositivo; (4) Login seguro por código.

📊 Fonte dos Dados

Dados extraídos do SIGAA (Sistema Integrado de Gestão de Atividades Acadêmicas) da UNEMAT. Os registros abrangem projetos de pesquisa, ações de extensão e programas de pós-graduação stricto sensu com dados a partir de 2024.
